# laravel関連のコマンド

カレントディレクトリにlaravelをインストールしてプロジェクトを自動生成
（プロジェクトを作るたびにlaravelをインストールする必要がある）
composer create-project laravel/laravel projectname

app/Http/Controllers ディレクトリに TaskController.php を作成。
php artisan make:controller TaskController 

foldersテーブルのマイグレーションファイルを作成する
php artisan make:migration create_folders_table --create=folders



.env
DB_CONNECTION=mysql

DB_PORT=3306
DB_DATABASE=first_db

DB_PASSWORD=
DB_CHARSE=utf8mb4          # 追記：文字コード設定

マイグレーションファイルを実行してテーブルを作成する
php artisan migrate



モデルを作成する
php artisan make:model Folder



# シーダーを作成する
php artisan make:seeder FoldersTableSeeder
