■venv(標準ライブラリ)
仮想環境の作成
exe化したいファイルがあるディレクトリで
python -m venv env
仮想環境の有効化
env\scripts\activate.ps1
仮想環境の無効化
deactivate
モジュールをファイルに記載して管理
pip freeze > requirements.txt
モジュールのインストール
pip install -r requirements.txt

envフォルダを削除するだけ
