# JavaScriptの実行タイミング
## 1. すぐに実行
処理がそのままベタ書きされている場合、すぐに実行される。  
※即時関数という便利テクニックがある。
```js
// 即時関数
// 無名関数をグループ化演算子()で囲み、さらに()を付けて実行する。
(() => {})();

// ※一回だけ実行する処理を実装するために使う。
// ※一回しか実行しない関数なので、再利用のための名前を付ける必要がない。
// ※むしろ名前を付けてしまうとグローバルスコープを汚染してしまう。
// ブックマークレットなどでよく使う。
```
